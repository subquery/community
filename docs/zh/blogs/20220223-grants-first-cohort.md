# SubQuery宣布第一批资助计划获得者

![](https://miro.medium.com/max/1400/1*qp0hhPcvodDIMmVScohSnw.png)

Polkadot领先的数据索引解决方案SubQuery很高兴地宣布，有四个团队被选为有史以来第一批SubQuery资助计划的获得者。 这些团队是 [Alphabit](https://www.polkadata.xyz/), [Zoombies](https://zoombies.world), [Dapplooker](https://dapplooker.com/) 和 [Dotsama](http://dotsama.ai/)。

50万美金的SubQuery资助计划于2021年12月启动，其目标是为区块链开发者，社区成员和更广泛的生态系统参与者提供机会，创造创新的或高潜力的想法，利用SubQuery的开源数据索引工具来支持前沿的去中心化的应用。

第一位是Alphabit，通过提供跟踪和分析基金公司，大型交易商或“whales”巨型交易商的投资组合情况和收益可预见性，努力赋予独立投资者权力。 他们使用SubQuery为所需的链上数据指标提供支持，推动他们成为Polkadot领先的市场分析公司。 SubQuery认为，这将对Polkadot生态系统产生重大影响，目前该生态系统缺乏强大的投资分析工具。

> “我们很荣幸也很高兴成为SubQuery资助计划的一份子，并与SubQuery建立合作伙伴关系。 SubQuery提供的商业建议和技术支持完全超出了我们作为小型分析初创公司所能获得的支持预期。 SubQuery提供的慷慨资金将允许我们为Polkadot/Kusama DeFi社区掀起下一波关键分析的浪潮。 这个项目最有价值的地方之一就是与SubQuery的战略合作伙伴和构建伙伴建立关系。 我们的旅程还很早，SubQuery的资助合作伙伴关系将是我们快速进入下一阶段所需的燃料。”---Michael Anderson，Alphabit商业发展与战略合作

第二位，Zoombies提交了一份提案，希望利用SubQuery加快实现他们的目标，即深入增加正在增长的，可收集play-to-earn通证的价值。 他们的项目可以访问游戏中对于已经过去的事件的完整历史记录，用于创造有意义的视图，丰富玩家体验。 SubQuery项目以及从SubQuery资助计划中获得的赠款将帮助Zoombies向用户提供与众不同信息丰富的前端。

> “SubQuery Grants计划是一个非常明确的逐步奖励驱动的过程。 James和团队指导我们采用迭代方法来实施，在尝试实施我们罗列的更具挑战性的任务之前，先提供可以实现的目标。 这种方法极大地推动了我们对项目成功的信心。”---Ryan Price，Zoombies首席执行官

我们初始组群中的第三支队伍是 [Dapplooker](https://dapplooker.com/)， 它旨在增强Polkadot项目的能力，使其拥有美丽和易于构建的分析工具，如探索、图表和仪表板。 这些有价值的指标将建立在SubQuery项目之上，以获取所需的数据，并以直观和美观的格式呈现给受众。

> _"Dapplooker 与 SubQuery 的伙伴关系将启用 Polkadot 开发者的分析和仪表板解决方案。 Polkadot用户可以在 SubQuery 数据顶部创建、叉和共享仪表板，以跟踪关键计量并提供关于性能和采用情况的反馈。 Dapplooker的直观视觉SQL编辑器将帮助以表格形式浏览GraphQL数据，并使用易于使用的编辑器编写SQL查询。_ - Abhinav Singh- 创始人兼首席开发者(Dapplooker)

最后，这个群组的第四个也是最后一个赠款接受者是 [Dotsama](http://dotsama.ai/)。 Dotsama团队还决心为Polkadot/Kusama生态系统中的普通投资者带来丰富的见解，目标是成为一家一站式分析商店，提供工具，识别早期投资趋势和机会。 为了使这种见解具有实力，Dotsama需要一个强有力的数据基础，它是SubQuery作为自然伙伴进入的地方。

> _“SubQuery及其生态系统的成功对于Polkadot和整个web3空间的成功非常重要。 _作为SubQuery资助计划的一部分，我们不仅希望获得技术和战略指导，我们还希望成为这个社区的一部分，帮助加速这一运动。”

管理SubQuery赠款方案的Brittany Seales评论， _“我们很高兴欢迎这些团队在获得有史以来第一笔SubQuery赠款之后进入SubQuery生态系统。 SubQuery 是Polkadot基础设施的一个基本部分，我们相信支持Zoombies, Alphabit, Dotsama 和 Dapplooker 可以将它们连接到现有的伞状团队，我们可以加速我们的 Web3 数据革命视野。”_

SubQuery渴望支持所有小组使他们的理想化为生，并希望这些小组的成功能够促使更多的人提出申请。 除了授予我们第一批团队的资助外，SubQuery网站上还提供了与Polkadot领先的平行链团队合作的持续技术资助，如Moonbeam、Acala、Astar和Bifrost。

我们鼓励对这些奖金感兴趣的申请人提交建议书，并附上完成任务所需的预算和其他要求。 如果你有任何问题，请加入我们的Discord，在Grants频道与我们的团队联系。

对于那些想在申请前提高SubQuery技能的人，[SubQuery学院](./20211018-subquery-launches-the-subquery-academy.md)的 "英雄课程 "是一个模块化教育计划，旨在提高开发人员的意识。 它是[完全免费并且可以在线上随时学习](https://subquery.coassemble.com/unlock/dOKZW6O#/)。

## 关于SubQuery：

[SubQuery](https://subquery.network)是一个去中心化的的数据聚合，索引和查询层，存在于Layer-1区块链和去中心化的应用程序之间。 提供解锁区块链数据并将其转换为可查询状态的服务，以便使用者能够用直观地使用。 SubQuery的协定是通过SubQuery SDK提取出区块链数据的特性，允许开发人员专注于部署其核心产品，而无需在定制后端技术上浪费精力。 该协议通过SubQuery Sdk 抽象了区块链数据特性，允许开发人员专注于部署他们的核心产品，而不必在后端技术上浪费不必要的精力。
