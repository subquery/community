## 关于Parallel Finance

Parallel Finance是一种分散的货币市场协议，在 Polkadot 生态系统中提供借贷、抵押和借款。 类似于Polkadot与其“先行网”Kusama之间的关系 Heiko Finance是通往Parallel的姐妹网络，我们希望在Kusama区块链上启动该网络。 我们正在建设一个权力下放的未来，使社区有能力提高资本效率。 我们通过杠杆动作和拍卖出借平台提供安全保障和普及。
